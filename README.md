## About
- Includes user register, login.
- Users can add link their profile and edit them.
- Users can obtain a password reset token to their email by using the "forgot my password" page. 

## Used

- .NET Core 6.0
- .NET Core 6.0 Identity
- AWS SES
- AWS RDS
- AWS S3
- Bootstrap
